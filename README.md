<h1 align="center">Adminer</h1>

This package provides REST API starterpack of admin management with no intervention to codebase and keep clean.

Getting Started
---

Installation :

```
$ composer require tripteki/laravelphp-adminer
```

Optionally how to use :

- Publish tests file into your project's directory with running :

```
php artisan vendor:publish --tag=tripteki-laravelphp-adminer-tests
```

- Generate swagger files into your project's directory with putting this into your annotation configuration :

```
base_path("vendor/tripteki/laravelphp-adminer/src")
```

Usage
---

`php artisan adminer:install`
